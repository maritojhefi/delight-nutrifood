Offline Detection

Detect when your users go offline and notify them. This is great if you are using your page in areas where connection is poor, and you want your users to always be informed of their connection status.

How does it work?

Simple. On this page, turn off WiFi / Data connection. A message will appear notifying you that you are now offline. Also, the message below will change.

Simulate Offline
Simulate Online
Disable Page

There maybe cases where you would like to completely remove access from a page if that page is offline. You can do that as well by showing a message prompting the user to connect.

Simulate Page Disabled
Where can I use it?

This function is natively integrated in our code. All pages will detect if you are online or offline and display this message. Turn off the internet on any page and the message will pop-up.
